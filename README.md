<br>

Clone the project

```bash
  git clone https://github.com/Caioscg/ToDoList...
```

Navigate to the project directory

```bash
  cd ToDoList
```

Install the dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
